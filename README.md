### Install:

`npm i`

### Build all files.

This command creates CSS from SASS, builds and minifies JS:

`npm run build`

### Build CSS from Sass:

`npm run sass`

### Build JS:

`npm run js`

### Start Server:

This command also runs mocks

`npm run server`

Stop Server:

`contol + c`

### Build HTML

Build all html files:

`npm run templates`

Watch & Build all html files:

`npm run watch-templates`
